Equivalentes de Comandos CMD en PowerShell

Comando CMD Equivalente en PowerShell Descripción
ipconfig Get-NetIPAddress / Get-NetIPConfiguration Muestra la configuración de red.
ping Test-Connection Verifica la conectividad con otro dispositivo.
tracert Test-NetConnection -TraceRoute Realiza un seguimiento de la ruta de red.
dir Get-ChildItem Lista archivos y directorios.
cd Set-Location Cambia el directorio actual.
copy Copy-Item Copia archivos o carpetas.
move Move-Item Mueve archivos o carpetas.
del Remove-Item Elimina archivos o carpetas.
cls Clear-Host Limpia la pantalla de la consola.
mkdir New-Item -ItemType Directory Crea un nuevo directorio.
rmdir Remove-Item Elimina un directorio.
type Get-Content Muestra el contenido de un archivo.
echo Write-Host Escribe texto en la consola.
set /p Read-Host Solicita y lee entrada del usuario.
set Set-Variable / Get-Variable Define o muestra variables de entorno.
tasklist Get-Process Muestra una lista de procesos en ejecución.
taskkill Stop-Process Finaliza un proceso en ejecución.
whoami [System.Security.Principal.WindowsIdentity]::GetCurrent().Name Muestra el usuario actual.
attrib Get-ItemProperty / Set-ItemProperty Visualiza o cambia atributos de archivos.
netstat Get-NetTCPConnection Muestra conexiones de red activas.
shutdown Stop-Computer / Restart-Computer Apaga o reinicia el equipo.
